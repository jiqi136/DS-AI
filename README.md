### 操作指引
- **模型切换**：控制面板左侧顶部下拉菜单选择。
- **会话管理**：对话记录下方可删除本次对话记录与清除联系上下文。
- **隐藏纪录**：点击对话记录下方的“★”，隐藏当前对话纪录，只有输入正确密码，才能查看。
- **快速唤醒**：点击电脑系统的托盘通知区图标就能恢复程序界面。
