# AI生成图片优化步骤介绍

为了获得更理想的AI生成图像效果，建议不要直接使用简短的中文提示词生成图片。推荐采用以下优化步骤：

## 步骤

1.  **借用网页版AI完善生图提示词**
    *   先用中文构思并完善你的提示词：在免费AI网页版工具中，仔细描述你想要的画面内容，包括**主体、风格、背景、细节、光影和氛围**等要素。
    *   尽量具体明确，可参考或整合已有的优秀提示词思路。
    *   **提示词：**
        *   完善下文，变为AI生成图片详细提示词。
        *   `一个女孩和一条龙，很好看。`

2.  **修改和调整直至满意**
    *   通过多次生成和对比，逐步修正中文提示词，使其更准确地表达你的意图。

3.  **翻译成英文**
    *   将修改满意后的中文提示词借助网页AI工具翻译转化为英文。英文提示词通常能更精确地被AI识别，从而生成更符合预期的图像。
